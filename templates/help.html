1. Browse Genome

 The ESKtides provided a user-friendly browse mode in which users can browse by strains or phage types, in each table users can search strains accession, genome length, ORF number and download CDS protein fasta. Users can click the drop-down menu or click detail to see annotation detail.


2. Browse Peptides

 Currently, two activity level mode are provided in ESKtides: (A). High level and (B).Medium level. In High level, users can obtain activity score which is more than 0.9, while in Medium level, users can obtain activity score which is within 0.5-0.9.


3. Analysis

 Users can use online analysis Peptides activity Prediction: mining peptides and score it, Peptides Phylogenetic tree and calculating Physicochemical property


4. Statistics

 This page provides basic statistical charts including (A). ESKAPE strains summary, (B). ESKAPE phages summary, (C). ESKAPE strains derived peptides activities distribution, (D). ESKAPE phages derived peptides activities distribution.


5. Download

 In download page, users can download all biomarker information or download specific information.
